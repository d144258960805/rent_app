# 產品需求文件 (PRD)

**專案名稱**：逢甲租屋網 (Feng Chia Rental Platform)
**專案描述**：專為逢甲學生與周邊房東打造的租屋媒合平台，透過雙向信用機制與房東實名認證，提供防範詐騙、資訊透明、且無仲介費的租屋體驗。

---

## 1. 專案概述

### 背景與動機
為了解決逢甲學生在尋找校內外安全房源時常遇到的資訊落差與詐騙風險，本專案希望能讓學弟妹不再因過時的資訊而白跑一趟，讓租屋過程像選課一樣清晰簡單。

### 目標用戶
- **逢甲學生**：尋找校內外安全房源的主要族群。
- **房東**：欲刊登房源並尋找穩定租客的屋主。

### 核心價值主張
- **防範詐騙**：導入房東實名認證與房契/權狀核對，確定租屋者非二房東或詐騙。
- **理想篩選**：提供精準的標籤篩選，確保租屋人能找到理想中的房子。
- **資訊透明**：資料透明公開（例如：水費、電費、管理費等資訊）。
- **去仲介化**：提供雙方直接溝通管道，省去仲介費。
- **信用機制**：建立雙向評分系統與實名認證，違約則扣除積分，以維護租屋環境。

## 2. 功能需求

以下為系統核心功能，以使用者故事（User Story）格式描述：

| 編號 | 功能名稱 | 使用者故事 |
|---|---|---|
| **F-01** | 分類篩選 | **作為** 學生租客，**我希望** 能夠依據租金範圍、房型、房屋大小、是否有租補等條件進行篩選，**以便** 節省時間成本，更有效率找到適合的房子。 |
| **F-02** | 關鍵字搜尋 | **作為** 學生租客，**我希望** 能透過 #標籤 搜索特定需求（如：落地窗、採光好、乾濕分離、養寵物），**以便** 縮小搜索範圍。 |
| **F-03** | 評論上傳 | **作為** 學生租客，**我希望** 在賞屋後能匿名上傳實際圖片或留下評論，**以便** 提供其他尋房學生真實的參考資訊。 |
| **F-04** | 房東身分驗證 | **作為** 系統管理者，**我希望** 房東在刊登前必須上傳證明文件（如身分證、權狀影本）並由後端審核，**以便** 防止虛假房源與詐騙。 |
| **F-05** | 雙向信用評分機制 | **作為** 平台使用者（房客/房東），**我希望** 系統具備實名積分機制（準時交租/快速維修可加分，惡意違約則扣分），**以便** 建立雙方互信的租屋環境。 |
| **F-06** | 揪團找室友區 | **作為** 逢甲學生，**我希望** 可以在平台發布「徵室友」資訊，且限定逢甲信箱認證者留言，**以便** 安全地為分租套房或整層住家尋找合適室友。 |
| **F-07** | 積分限制存取 | **作為** 系統管理者，**我希望** 當使用者積分低於門檻（如 80 分）時，能自動限制其看房預約或刊登權限，**以便** 維護平台品質。 |

## 3. 非功能需求

### 技術限制
- **前端**：HTML / CSS / JavaScript（不使用任何前端框架，依課堂規範）
- **後端**：Python + Flask（依課堂規範）
- **資料庫**：SQLite（輕量化、易於移植）
- **外部串接**：Google Maps API（用於實作地圖定位功能）

### 效能與安全性考量
- **效能**：
  - 租屋旺季（5-7月）需能支援至少 500 名使用者同時在線而不崩潰。
  - 頁面加載時間需控制在 2 秒內，且房源圖片須進行壓縮優化。
- **安全性**：
  - 使用者密碼需經過雜湊（Hashing）加密儲存。
  - 涉及個人資訊（身分證字號、手機號碼等）需符合資安規範，敏感證件資料於驗證後即刪除，並限制僅管理者可查看。
  - 房東端身分驗證機制，防止虛假房源。
- **易用性**：
  - 介面符合響應式網頁設計（RWD），方便學生使用行動裝置瀏覽。
  - 地圖搜尋功能可快速定位距離「逢甲校門」或「商圈中心」的距離。
- **可靠性**：
  - 系統年度可用性需達 99.5%，並具備每日自動備份機制以確保租賃合約資料不遺失。

## 4. MVP 範圍

為確保專案能順利進行展示，系統功能依優先級劃分如下：

### Must Have (核心必要功能)
- 房東與學生會員註冊 / 登入機制
- 房東身分驗證與文件審核 (F-04)
- 房源刊登與首頁清單展示
- 房源分類篩選 (F-01)
- 雙向信用評分機制基礎架構 (F-05)

### Should Have (重要期望功能)
- 關鍵字標籤搜尋 (F-02)
- 匿名評論與實際圖片上傳 (F-03)
- 揪團找室友公告欄（含逢甲信箱認證） (F-06)
- 積分低於門檻限制預約或刊登權限 (F-07)

### Nice to Have (額外加分功能)
- 整合 Google Maps API 進行地圖距離計算

*(註：範圍限制外，本次不開發自動產生虛擬帳號、線上即時簽約、室內導航功能，且房源數據僅限於逢甲大學周邊。)*
