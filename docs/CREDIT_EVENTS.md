# 雙向信用評分機制 (F-05) - 積分事件表

本文件定義逢甲租屋網中所有與信用積分相關的觸發事件與增減數值。
預設每位通過實名認證的用戶（包含房東與房客），初始積分為 **100 分**。

## 觸發事件與積分增減規則

| 事件代碼 (Event Key) | 事件說明 | 積分異動 | 適用對象 |
| -------------------- | -------- | -------- | -------- |
| `account_created` | 帳號建立 (未認證) | 50 | 所有人 |
| `landlord_verified` | 房東身分驗證通過 | +50 | 房東 |
| `tenant_verified` | 學生身分(逢甲信箱)驗證通過 | +50 | 房客 |
| `rent_paid_on_time` | 準時繳交租金 | +5 | 房客 |
| `maintenance_quick` | 快速處理維修需求 | +5 | 房東 |
| `review_submitted` | 發表優質/真實賞屋評論 | +2 | 房客 |
| `rent_delayed` | 遲繳租金 | -10 | 房客 |
| `fake_property` | 刊登虛假房源被檢舉核實 | -30 | 房東 |
| `contract_breach_malicious` | 惡意違約 | -50 | 所有人 |

## 系統存取限制
* **< 80 分**：限制房客預約看房、限制房東刊登新房源。
* **< 60 分**：列入黑名單，介面顯示紅色警示標籤，且強制限制多數平台互動功能。
